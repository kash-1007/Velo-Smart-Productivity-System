# Velo-Smart-Productivity-System
An AI-powered personal and professional productivity platform that helps users organize tasks, prioritize work intelligently, analyze performance, and boost efficiency through real-time insights and automation.

Tech Stack
**Frontend**
1.Angular 17
2.Angular Material
3.RxJS
4.TypeScript
**Backend**
1.Python
2.FastAPI
3.SQLAlchemy
4.PostgreSQL
**ML / AI**
1.PyTorch
2.Sentence Transformers
3.Classification Models
4.LLM-based text extraction (future)
**DevOps**
1.Docker
2.GitHub Actions
3.Render / Netlify
4.Neon PostgreSQL (free tier)
